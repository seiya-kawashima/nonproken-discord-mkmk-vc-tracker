# ========================================
# 本番環境用 依存パッケージ定義ファイル (requirements.txt)
# ========================================
# 概要: このプログラムを実行するために必要な外部ライブラリのリスト
#       pipコマンドでこのファイルを読み込むと、必要なライブラリが自動インストールされる
#
# なぜこのファイルが必要？:
#   - プロジェクトで使用する外部ライブラリを明確に記録
#   - 開発環境を他の人と共有する際に、同じバージョンのライブラリをインストール可能
#   - 本番環境へのデプロイ時に必要なライブラリを自動インストール
#   - セキュリティ脆弱性のあるバージョンを避けるため、バージョンを固定
#
# 使い方:
#   pip install -r requirements.txt
#
# Python バージョン要件: 3.11以上
# ========================================
