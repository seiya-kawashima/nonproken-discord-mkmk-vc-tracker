# ========================================
# 開発環境用 依存パッケージ定義ファイル (requirements-dev.txt)
# ========================================
# 概要: 開発者がローカル環境で開発する際に必要な全てのライブラリ
#       本番用、テスト用、開発専用ツールを全て含む完全セット
#
# なぜこのファイルが必要？:
#   - 開発者が必要な全てのツールを一度にインストールできる
#   - ドキュメント生成、デバッグツール、開発補助ツールを含む
#   - 新しい開発者がすぐに開発を始められる環境を提供
#   - コードを書く前の準備（pre-commit）などの自動化ツールを含む
#
# 使い方:
#   pip install -r requirements-dev.txt
#   (これだけで開発に必要な全てがインストールされる)
#
# 含まれる内容:
#   - requirements.txt (本番用)
#   - requirements-tst.txt (テスト用)
#   - 開発専用ツール
# ========================================
